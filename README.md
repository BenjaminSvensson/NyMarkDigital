# NyMarkDigital

Statisk hemsida för NyMarkDigital med portföljen som huvudfokus.

## Struktur

- `index.html` - startsidan med portfolio, process och kontakt.
- `styles.css` - Swiss-inspirerad visuell riktning med vit yta, gridlinjer och blå accent.
- `script.js` - filtrering av portföljkort.
- `examples/` - interna hemsideexempel som kan visas direkt från portföljen.
- `assets/nymarkdigital-logo.png` - logotypen fran den bifogade bilden.

## Lägg till portfolioobjekt

1. Skapa en ny katalog under `examples/` eller använd en extern URL.
2. Kopiera ett befintligt `.work-card` i `index.html`.
3. Uppdatera `href`, namn, kategori och beskrivning.

Sidan kan oppnas direkt som statisk HTML eller hostas via valfri statisk hosting.
